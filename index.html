Projetos

Projeto-Iframe

Projeto desenvolvido para incorporar conteúdo externo de forma integrada à página, utilizando a tag HTML iframe. O foco foi garantir que o elemento se adaptasse ao layout, mantendo boa experiência de navegação e responsividade, além de aplicar ajustes no CSS para eliminar margens e bordas, criando uma integração visual limpa e contínua.

Projeto Login

Uma página de login moderna e responsiva, onde a imagem se ajusta dinamicamente à tela. Conforme o tamanho e a orientação do dispositivo mudam, ela se move para o canto mais apropriado, garantindo equilíbrio visual e foco no formulário. As transições são suaves, criando uma experiência fluida e agradável para o usuário.
