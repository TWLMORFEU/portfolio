Projetos

Projeto Iframe

Projeto com integração de conteúdo externo via iframe e botões interativos com efeito de translate ao passar o mouse, proporcionando uma navegação dinâmica e moderna.

Projeto Login

Um formulário de login simples, mas estilizado. Neste projeto, utilizei HTML e CSS para criar uma tela de autenticação com design limpo e moderno. É um exemplo básico de estrutura de formulário com foco em design responsivo.

Projeto Fanta

Projeto com efeito de scroll interativo, onde imagens se deslocam dinamicamente conforme a rolagem da página, criando uma experiência visual envolvente e intuitiva.

Projeto Cordel

Este projeto demonstra o uso de imagens com efeito de parallax, proporcionando uma rolagem com animação fluida e agradável ao usuário. A estrutura é composta por seções com textos centralizados, uso controlado de espaçamento, e estilização de fontes personalizadas. Ele foi desenvolvido com foco em praticar o posicionamento de elementos na tela e o uso de recursos visuais em CSS puro, sem a utilização de frameworks ou bibliotecas externas.

Projeto Android

Página inicial simples desenvolvida como primeiro exercício prático, aplicando conceitos básicos de HTML e CSS. O projeto apresenta estruturação semântica, uso de cores, textos, títulos e elementos visuais, representando o início da jornada no desenvolvimento front-end.
