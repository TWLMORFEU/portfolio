Meus Projetos

Importante:
Os projetos apresentados aqui não têm como objetivo principal serem esteticamente perfeitos ou visualmente sofisticados, mas sim demonstrar as habilidades técnicas e os conceitos que domino. Cada projeto foi desenvolvido para evidenciar competências específicas, práticas e conhecimentos que aplico na criação de soluções funcionais e eficientes.

Projeto Iframe

Projeto desenvolvido para incorporar conteúdo externo de forma integrada à página, utilizando a tag HTML iframe. O foco foi garantir que o elemento se adaptasse ao layout, mantendo boa experiência de navegação e responsividade, além de aplicar ajustes no CSS para eliminar margens e bordas, criando uma integração visual limpa e contínua.

Projeto Login

Uma página de login moderna e responsiva, onde a imagem se ajusta dinamicamente à tela. Conforme o tamanho e a orientação do dispositivo mudam, ela se move para o canto mais apropriado, garantindo equilíbrio visual e foco no formulário. As transições são suaves, criando uma experiência fluida e agradável para o usuário.

Projeto Cordel

Página estática que apresenta um cordel em seções, com uma imagem de fundo em parallax discreto para dar leve sensação de profundidade sem distrair da leitura. Foco em tipografia, contraste e ritmo do texto.

Projeto Fanta

Página interativa inspirada na marca Fanta, onde as imagens descem suavemente ao rolar a página. O efeito cria uma sensação dinâmica, mas sem exageros, mantendo o foco visual nos produtos e cores da marca.
